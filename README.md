# Website Traffic Analysis

## Project Overview

This project analyzes website traffic and user interaction data to identify traffic patterns, user engagement, popular content, and geographical distribution.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features

- Event
- Date
- Country
- City
- Artist
- Album
- Track
- ISRC
- Link ID

## Analysis Performed

- Data Cleaning
- Exploratory Data Analysis
- Event Distribution Analysis
- Top Countries Analysis
- Top Cities Analysis
- Popular Artists Analysis
- Popular Albums Analysis
- Popular Tracks Analysis
- Daily Traffic Trend Analysis

## Key Insights

- Identified most active countries and cities.
- Determined top-performing artists and tracks.
- Found peak user activity periods.
- Identified most accessed links.

## Recommendations

1. Focus on high-traffic regions.
2. Promote popular content.
3. Improve engagement in low-performing regions.
4. Optimize frequently visited links.
5. Schedule campaigns during peak traffic periods.
